# 🤟 基于 YOLOv5 与 Spring Boot 3 的分布式手语识别系统

> **项目关键词**：Spring Boot 3, Vue 3, YOLOv5, Docker, Nginx, Redis, Micro-services

## 📖 项目简介

这是一个前后端分离的、支持容器化部署的实时手语识别系统。项目旨在解决听障人士的沟通障碍问题，通过计算机视觉技术实现手语到文本的实时转换。

**核心差异化优势**：
不同于传统的单体 Python 脚本，本项目采用 **工程化架构**。将 AI 推理服务（Python/Flask）与 业务逻辑服务（Java/Spring Boot）解耦，通过 Docker 容器编排管理，并引入 Redis 缓存与 Nginx 反向代理，构建了一个**高可用、低耦合、易于扩展**的现代化 Web 系统。

---

## 🛠 技术栈 (Tech Stack)

### 💻 后端 & 架构
- **核心框架**：Spring Boot 3.x (Java 17)
- **AI 推理**：Python 3.9 + Flask + YOLOv5 (ONNX Runtime 加速)
- **数据库**：MySQL 8.0
- **缓存中间件**：Redis 7.0 (Cache Aside Pattern)
- **网关/代理**：Nginx (反向代理 / 动静分离 / 负载均衡)

### 🎨 前端
- **框架**：Vue 3 (Composition API)
- **脚手架**：Vite
- **组件库**：Element Plus
- **网络请求**：Axios

### 🐳 运维 & 部署
- **容器化**：Docker & Docker Compose
- **操作系统**：Linux (Ubuntu/CentOS)
- **对象存储**：MinIO (自建 Docker 部署，用于存储视频/图片)

---

本系统采用 **微服务伴生模式 (Sidecar-like)**，Java 负责重业务逻辑，Python 负责重计算任务。
