# 虚拟手语交流平台

#### 介绍
前后端分离开发框架，基于 SpringBoot + Vue 的虚拟手语交流平台。

#### 软件架构
后端：SpringBoot + vue + mysql + redis


#### 安装教程

1.  执行sql脚本，创建数据库表
2.  修改 application.yml 配置文件，配置数据库连接信息
3.  启动 redis 服务
4.  启动后端项目 (XAdminApplication.java)
5.  启动前端项目 (cd vue-admin && npm run dev)
6.  访问 http://localhost:8080/
